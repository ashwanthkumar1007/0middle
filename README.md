# 0Middle

E-commerce application connecting farmers directly to consumers.

## Overview

0Middle is a platform that eliminates middlemen, allowing farmers to sell their produce directly to consumers at fair prices.

## Tech Stack

- Angular 18+
- SCSS for styling
- TypeScript
- Mobile-first, responsive design
- SSR/SSG compatible

## Getting Started

1. Install dependencies:

```bash
npm install
```

2. Run the development server:

```bash
npm start
```

3. Navigate to `http://localhost:4200/`

## Build

Run `npm run build` to build the project. The build artifacts will be stored in the `dist/` directory.

## Features

- **Landing Page**: Role selection (Farmer/Seller or Consumer/Buyer)
- **Static Application**: No backend integration initially
- **Accessible**: Keyboard navigation and screen reader support
- **Responsive**: Mobile-first design
